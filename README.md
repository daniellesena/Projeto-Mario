# 🔧 Mario Bros - Encanadores Profissionais

Um website criativo e divertido para um serviço de encanamento temático inspirado nos famosos encanadores do Reino dos Cogumelos - Mario e Luigi!

## 📋 Características

- **Cabeçalho Responsivo**: Navegação intuitiva com logo da marca
- **Design Atrativo**: Layout moderno com vídeo de fundo em fullscreen
- **Formulário de Contato Modal**: Pop-up elegante com animação suave para solicitação de orçamentos
- **Call-to-Action**: Múltiplas formas de contacto (formulário e WhatsApp)
- **Tema Temático**: Design colorido e divertido que remete ao universo Mario Bros
- **Integração com FormCarry**: Recebimento seguro de mensagens via email

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com animações e gradientes
- **JavaScript**: Interatividade (modal, formulário dinâmico)
- **FormCarry**: Serviço de envio de formulários
- **WhatsApp API**: Integração para contato direto

## 🎨 Funcionalidades Principais

✨ **Formulário Modal** - Abre/fecha com animação ao clicar em "Fale Conosco" ou "Entre em Contato"

📱 **Botão WhatsApp Flutuante** - Acesso rápido ao atendimento via WhatsApp

🎥 **Vídeo Background** - Vídeo em loop como fundo da página

📧 **Envio de Orçamentos** - Sistema integrado de contato com validação

## 🚀 Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Clique em "Fale Conosco" ou "Entre em Contato" para abrir o formulário
4. Preencha seus dados e solicite um orçamento
